*** Nova Lógica para busca de Massivas ***

Criar 2 Botões:
  - Busca na Mancha GPON
  - busca cidades atendidas
  - Busca na Mancha GPON e Busca cidades atendidas


Para os 2 habilitados:
  Busca primeiro na mancha GPON;
  Se não encontrar
  Buscar no raio de aproximado e nas cidades atendidadas
  Se não tiver raio
  Buscar nas cidades atendidas
  Se não estiver dentro do raio
  Buscar nas cidades atendidas

  Retornar nessa ordem

  Status,	Mancha GPON,	Dist. GPON (mts),	REDE PTP

Para busca nas cidades atendidas, as precisa ter as colunas de Cidade e UF

Modelo de busca por cidades

SELECT 
    c.nome AS cidade, 
    e.uf, 
    e.nome AS estado,
    c.latitude AS lat,
    c.longitude AS lon,
    (
        6371 * ACOS( # TESTE -> -11.618338691359725, -57.23432227375857
            COS(RADIANS(-23.1510513)) # Latitude
            * COS(RADIANS(c.latitude)) 
            * COS(RADIANS(c.longitude) - RADIANS(-45.7995311)) # Longitude
            + SIN(RADIANS(-23.1510513)) # Latitude
            * SIN(RADIANS(c.latitude))
        ) 
    ) AS distancia_km
FROM
	municipios c
INNER JOIN
    estados e ON UPPER(c.codigo_uf) = UPPER(e.codigo_uf)
HAVING 
	distancia_km <= 50
ORDER BY distancia_km ASC
LIMIT 4;
